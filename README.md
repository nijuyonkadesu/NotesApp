# NotesApp
App is designed by following the best architecture patterns. Built with love in KMM & Jetpack Compose. Same logic, Native UI, both iOS and Android in single codebase.

## Features
- [x] Data persitency
- [x] Supporting Search
- [x] SQL Delight
- [x] Date-Time Format
